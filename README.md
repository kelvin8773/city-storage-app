# City Store Order Tracking System

> Frontend challenge

## Getting Start

> Installation

```bash
 $ yarn install
```

> Start Develop / Demo

```bash
$ yarn start

# start web server only
$ yarn start:web

# start mock server only
$ yarn start:mock

```

> Code formating and lint check

```bash
# code formating
$ yarn format

# syntax errors check
$ yarn lint

# syntax errors fix
$ yarn lint:fix
```
